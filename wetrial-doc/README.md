## wetrial 文档

### nav

对应顶部菜单栏，排在第三位；配置如下；

| 配置项 | 说明               | 值   |
| ------ | ------------------ | ---- |
| title  | 前端文档           | doc  |
| order  | 在顶部菜单中的顺序 | 100  |
| path   | 路由的基础路径     | /doc |

### group

对应左边的左侧菜单栏分组

| order | title    | path      |
| ----- | -------- | --------- |
| 1     | 团队风格 | /standard |
| 2     | 设计规范 | /design   |
| 3     | 样式     | /style    |
| 4     | 教程     | /course   |
| 10    | 其他     | /other    |
